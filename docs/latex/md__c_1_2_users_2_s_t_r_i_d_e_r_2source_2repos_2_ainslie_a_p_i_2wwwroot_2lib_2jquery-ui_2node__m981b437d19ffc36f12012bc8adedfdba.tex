Recursively tests whether two values contains the same keys and values.

{\ttfamily callback} specifies the equality comparison function used to compare non-\/object values. It defaults to using the \`{}is()\`{} function.

If the values are both an object or array, it will recurse into both values, checking if their keys/values are equal. It will only check the keys and values contained by the objects; it will not check the objects\textquotesingle{} prototypes. If either of the values are not objects, they will be checked using the {\ttfamily callback} function.

{\bfseries{Esprima}} (\href{http://esprima.org}{\texttt{ esprima.\+org}}) is a high performance, standard-\/compliant \href{http://www.ecma-international.org/publications/standards/Ecma-262.htm}{\texttt{ ECMAScript}} parser written in ECMAScript (also popularly known as \href{http://en.wikipedia.org/wiki/JavaScript>JavaScript}{\texttt{ Java\+Script}}). Esprima is created and maintained by \href{http://twitter.com/ariyahidayat}{\texttt{ Ariya Hidayat}}, with the help of \href{https://github.com/ariya/esprima/contributors}{\texttt{ many contributors}}.

Esprima runs on web browsers (IE 6+, Firefox 1+, Safari 3+, Chrome 1+, Konqueror 4.\+6+, Opera 8+) as well as \href{http://nodejs.org}{\texttt{ Node.\+js}}.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m336af5ca0a8352395aeb06567d0974c5_autotoc_md4181}{}\doxysubsubsection{\texorpdfstring{Features}{Features}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m336af5ca0a8352395aeb06567d0974c5_autotoc_md4181}

\begin{DoxyItemize}
\item Full support for \href{http://www.ecma-international.org/publications/standards/Ecma-262.htm}{\texttt{ ECMAScript 5.\+1}}(ECMA-\/262)
\item Sensible \href{http://esprima.org/doc/index.html\#ast}{\texttt{ syntax tree format}} compatible with Mozilla \href{https://developer.mozilla.org/en/SpiderMonkey/Parser_API}{\texttt{ Parser AST}}
\item Heavily tested (\texorpdfstring{$>$}{>} 550 \href{http://esprima.org/test/}{\texttt{ unit tests}} with solid 100\% statement coverage)
\item Optional tracking of syntax node location (index-\/based and line-\/column)
\item Experimental support for ES6/\+Harmony (module, class, destructuring, ...)
\end{DoxyItemize}

Esprima is blazing fast (see the \href{http://esprima.org/test/benchmarks.html}{\texttt{ benchmark suite}}). It is up to 3x faster than Uglify\+JS v1 and it is still \href{http://esprima.org/test/compare.html}{\texttt{ competitive}} with the new generation of fast parsers.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m336af5ca0a8352395aeb06567d0974c5_autotoc_md4182}{}\doxysubsubsection{\texorpdfstring{Applications}{Applications}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m336af5ca0a8352395aeb06567d0974c5_autotoc_md4182}

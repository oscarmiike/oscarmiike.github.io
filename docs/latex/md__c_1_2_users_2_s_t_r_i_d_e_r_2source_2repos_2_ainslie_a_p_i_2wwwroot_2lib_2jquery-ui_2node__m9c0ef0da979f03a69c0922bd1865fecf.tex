\begin{DoxyItemize}
\item {\ttfamily path}\+: Use instead of the {\ttfamily PATH} environment variable.
\item {\ttfamily path\+Ext}\+: Use instead of the {\ttfamily PATHEXT} environment variable.
\item {\ttfamily all}\+: Return all matches, instead of just the first one. Note that this means the function returns an array of strings instead of a single string.

Create a new directory and any necessary subdirectories at {\ttfamily dir} with octal permission string {\ttfamily opts.\+mode}. If {\ttfamily opts} is a non-\/object, it will be treated as the {\ttfamily opts.\+mode}.

If {\ttfamily opts.\+mode} isn\textquotesingle{}t specified, it defaults to {\ttfamily 0777 \& (\texorpdfstring{$\sim$}{\string~}process.umask())}.

{\ttfamily cb(err, made)} fires with the error or the first directory {\ttfamily made} that had to be created, if any.

You can optionally pass in an alternate {\ttfamily fs} implementation by passing in {\ttfamily opts.\+fs}. Your implementation should have {\ttfamily opts.\+fs.\+mkdir(path, mode, cb)} and {\ttfamily opts.\+fs.\+stat(path, cb)}.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m8fab29ed05ce700fc72672f652343c55_autotoc_md2375}{}\doxysubsection{\texorpdfstring{mkdirp.\+sync(dir, opts)}{mkdirp.\+sync(dir, opts)}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m8fab29ed05ce700fc72672f652343c55_autotoc_md2375}
Synchronously create a new directory and any necessary subdirectories at {\ttfamily dir} with octal permission string {\ttfamily opts.\+mode}. If {\ttfamily opts} is a non-\/object, it will be treated as the {\ttfamily opts.\+mode}.

If {\ttfamily opts.\+mode} isn\textquotesingle{}t specified, it defaults to {\ttfamily 0777 \& (\texorpdfstring{$\sim$}{\string~}process.umask())}.

Returns the first directory that had to be created, if any.

You can optionally pass in an alternate {\ttfamily fs} implementation by passing in {\ttfamily opts.\+fs}. Your implementation should have {\ttfamily opts.\+fs.\+mkdir\+Sync(path, mode)} and {\ttfamily opts.\+fs.\+stat\+Sync(path)}.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m8fab29ed05ce700fc72672f652343c55_autotoc_md2376}{}\doxysection{\texorpdfstring{usage}{usage}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m8fab29ed05ce700fc72672f652343c55_autotoc_md2376}

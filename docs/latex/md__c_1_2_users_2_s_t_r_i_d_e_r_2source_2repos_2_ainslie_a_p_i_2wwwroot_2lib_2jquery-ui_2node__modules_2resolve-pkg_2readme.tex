 \begin{quote}
Resolve the path of a package regardless of it having an entry point \end{quote}
Some packages like CLI tools and grunt tasks don\textquotesingle{}t have a entry point, like {\ttfamily "{}main"{}\+: "{}foo.\+js"{}} in package.\+json, resulting in them not being resolvable by {\ttfamily require.\+resolve()}.

Parses an \href{http://en.wikipedia.org/wiki/Iso8601}{\texttt{ ISO8601}} date and returns the number of milliseconds since January 1, 1970, 00\+:00\+:00 UTC, or {\ttfamily NaN} if it is not a valid ISO8601 date.

This parses {\itshape all} ISO8601 dates, including dates without times, \href{https://en.wikipedia.org/wiki/ISO_8601\#Ordinal_dates}{\texttt{ ordinal dates}}, and the compact representation (omitting delimeters). The only exception is \href{https://en.wikipedia.org/wiki/ISO_week_date}{\texttt{ ISO week dates}}, which are not parsed.

If no time zone offset is specified, it assumes UTC time.

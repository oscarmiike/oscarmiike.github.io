Therefore a lone negation (e.\+g. `\mbox{[}'!foo\textquotesingle{}\mbox{]}{\ttfamily ) will never match anything – use}\mbox{[}\textquotesingle{}\texorpdfstring{$\ast$}{*}\textquotesingle{}, \textquotesingle{}!foo\textquotesingle{}\mbox{]}\`{} instead.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__modules_2multimatch_2readme_autotoc_md10378}{}\doxysection{\texorpdfstring{Globbing patterns}{Globbing patterns}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__modules_2multimatch_2readme_autotoc_md10378}

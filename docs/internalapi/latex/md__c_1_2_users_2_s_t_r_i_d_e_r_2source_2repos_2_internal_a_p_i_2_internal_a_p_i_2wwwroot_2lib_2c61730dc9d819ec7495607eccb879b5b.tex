 \begin{quote}
Strip comments from JSON. Lets you use comments in your JSON files! \end{quote}
This is now possible\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\{}
\DoxyCodeLine{\ \ \ \ //\ rainbows}
\DoxyCodeLine{\ \ \ \ "{}unicorn"{}:\ /*\ ❤\ */\ "{}cake"{}}
\DoxyCodeLine{\}}

\end{DoxyCode}


It will remove single-\/line comments {\ttfamily //} and mult-\/line comments {\ttfamily /\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}/}.

Also available as a \href{https://github.com/sindresorhus/gulp-strip-json-comments}{\texttt{ gulp}}/\href{https://github.com/sindresorhus/grunt-strip-json-comments}{\texttt{ grunt}}/\href{https://github.com/sindresorhus/broccoli-strip-json-comments}{\texttt{ broccoli}} plugin and a \href{https://github.com/uTest/autostrip-json-comments}{\texttt{ require hook}}.

{\itshape There\textquotesingle{}s already \href{https://npmjs.org/package/json-comments}{\texttt{ json-\/comments}}, but it\textquotesingle{}s only for Node.\+js and uses a naive regex to strip comments which fails on simple cases like {\ttfamily \{"{}a"{}\+:"{}//"{}\}}. This module however parses out the comments.}\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_2c61730dc9d819ec7495607eccb879b5b_autotoc_md4406}{}\doxysection{\texorpdfstring{Install}{Install}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_2c61730dc9d819ec7495607eccb879b5b_autotoc_md4406}

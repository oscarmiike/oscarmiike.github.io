Return an array of concatenated elements by calling {\ttfamily fn(x, i)} for each element {\ttfamily x} and each index {\ttfamily i} in the array {\ttfamily xs}.

When {\ttfamily fn(x, i)} returns an array, its result will be concatenated with the result array. If {\ttfamily fn(x, i)} returns anything else, that value will be pushed onto the end of the result array.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_2de393de978b2d3d1651c62f85b344876_autotoc_md2077}{}\doxysection{\texorpdfstring{install}{install}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_2de393de978b2d3d1651c62f85b344876_autotoc_md2077}

The {\ttfamily logger} is optional and is expected to be an instance of the bower \href{https://github.com/bower/logger}{\texttt{ logger}}. ~\newline
 Available constructor options\+:


\begin{DoxyItemize}
\item {\ttfamily cache}\+: the cache folder to use for some operations; using null will disable persistent cache (defaults to bower registry cache folder)
\item {\ttfamily registry.\+search}\+: an array of registry search endpoints (defaults to the Bower server)
\item {\ttfamily registry.\+register}\+: the endpoint to use when registering packages (defaults to the Bower server)
\item {\ttfamily registry.\+publish}\+: the endpoint to use when publishing packages (defaults to the Bower server)
\item {\ttfamily ca.\+search}\+: an array of CA certificates for each registry.\+search (defaults to null).
\item {\ttfamily ca.\+register}\+: the CA certificate for registry.\+register
\item {\ttfamily ca.\+publish}\+: the CA certificate for registry.\+publish
\item {\ttfamily proxy}\+: the proxy to use for http requests (defaults to null)
\item {\ttfamily https\+Proxy}\+: the proxy to use for https requests (defaults to null)
\item {\ttfamily strict\+Ssl}\+: whether or not to do SSL key validation when making requests via https (defaults to true).
\item {\ttfamily user\+Agent}\+: the user agent to use for the requests (defaults to null)
\item {\ttfamily timeout}\+: the timeout for the requests to finish (defaults to 60000)
\item {\ttfamily force}\+: If set to true, cache will be bypassed and remotes will always be hit (defaults to false).
\item {\ttfamily offline}\+: If set to true, only the cache will be used (defaults to false).
\end{DoxyItemize}

Note that {\ttfamily force} and {\ttfamily offline} are mutually exclusive. The cache will speedup operations such as {\ttfamily list}, {\ttfamily lookup} and {\ttfamily search}. Different operations may have different cache expiration times.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_2230f9f1596530d093c232d853f38b7f6_autotoc_md5348}{}\doxysubsubsection{\texorpdfstring{.lookup(name, callback)}{.lookup(name, callback)}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_2230f9f1596530d093c232d853f38b7f6_autotoc_md5348}

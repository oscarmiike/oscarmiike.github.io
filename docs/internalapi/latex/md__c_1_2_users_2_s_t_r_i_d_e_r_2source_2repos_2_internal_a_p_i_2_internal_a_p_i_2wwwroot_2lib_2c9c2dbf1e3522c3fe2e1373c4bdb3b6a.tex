{\ttfamily base64-\/js} does basic base64 encoding/decoding in pure JS.

\href{http://travis-ci.org/beatgammit/base64-js}{\texttt{ }}

Many browsers already have base64 encoding/decoding functionality, but it is for text data, not all-\/purpose binary data.

Sometimes encoding/decoding binary data in the browser is useful, and that is what this module does.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_2c9c2dbf1e3522c3fe2e1373c4bdb3b6a_autotoc_md554}{}\doxysection{\texorpdfstring{install}{install}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_2c9c2dbf1e3522c3fe2e1373c4bdb3b6a_autotoc_md554}

{\ttfamily base64js} has three exposed functions, {\ttfamily byte\+Length}, {\ttfamily to\+Byte\+Array} and {\ttfamily from\+Byte\+Array}, which both take a single argument.


\begin{DoxyItemize}
\item {\ttfamily byte\+Length} -\/ Takes a base64 string and returns length of byte array
\item {\ttfamily to\+Byte\+Array} -\/ Takes a base64 string and returns a byte array
\item {\ttfamily from\+Byte\+Array} -\/ Takes a byte array and returns a base64 string

 \begin{quote}
Get the visual width of the widest line in a string -\/ the number of columns required to display it \end{quote}
Some Unicode characters are \href{https://en.wikipedia.org/wiki/Halfwidth_and_fullwidth_forms}{\texttt{ fullwidth}} and use double the normal width. \href{http://en.wikipedia.org/wiki/ANSI_escape_code}{\texttt{ ANSI escape codes}} are stripped and doesn\textquotesingle{}t affect the width.

Useful to be able to know the maximum width a string will take up in the terminal.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_2c05a8e0db6aac173f1f69d598395a7f6_autotoc_md5239}{}\doxysection{\texorpdfstring{Install}{Install}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_2c05a8e0db6aac173f1f69d598395a7f6_autotoc_md5239}

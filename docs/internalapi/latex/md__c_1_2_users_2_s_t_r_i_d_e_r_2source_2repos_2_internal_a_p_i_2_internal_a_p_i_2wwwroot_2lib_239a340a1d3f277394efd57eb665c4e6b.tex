{\bfseries{Esprima}} (\href{http://esprima.org}{\texttt{ esprima.\+org}}, BSD license) is a high performance, standard-\/compliant \href{http://www.ecma-international.org/publications/standards/Ecma-262.htm}{\texttt{ ECMAScript}} parser written in ECMAScript (also popularly known as \href{https://en.wikipedia.org/wiki/JavaScript}{\texttt{ Java\+Script}}). Esprima is created and maintained by \href{https://twitter.com/ariyahidayat}{\texttt{ Ariya Hidayat}}, with the help of \href{https://github.com/jquery/esprima/contributors}{\texttt{ many contributors}}.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_239a340a1d3f277394efd57eb665c4e6b_autotoc_md8148}{}\doxysubsubsection{\texorpdfstring{Features}{Features}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_239a340a1d3f277394efd57eb665c4e6b_autotoc_md8148}

\begin{DoxyItemize}
\item Full support for ECMAScript 2017 (\href{http://www.ecma-international.org/publications/standards/Ecma-262.htm}{\texttt{ ECMA-\/262 8th Edition}})
\item Sensible \href{https://github.com/estree/estree/blob/master/es5.md}{\texttt{ syntax tree format}} as standardized by \href{https://github.com/estree/estree}{\texttt{ ESTree project}}
\item Experimental support for \href{https://facebook.github.io/jsx/}{\texttt{ JSX}}, a syntax extension for \href{https://facebook.github.io/react/}{\texttt{ React}}
\item Optional tracking of syntax node location (index-\/based and line-\/column)
\item \href{http://esprima.org/test/ci.html}{\texttt{ Heavily tested}} (\texorpdfstring{$\sim$}{\string~}1500 \href{https://github.com/jquery/esprima/tree/master/test/fixtures}{\texttt{ unit tests}} with \href{https://codecov.io/github/jquery/esprima}{\texttt{ full code coverage}})

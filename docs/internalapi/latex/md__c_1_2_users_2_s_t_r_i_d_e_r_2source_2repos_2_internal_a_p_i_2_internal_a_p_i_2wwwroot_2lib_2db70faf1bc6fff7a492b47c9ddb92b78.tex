Kurento Json RPC.

The Kurento jsonrpc project is a small RPC library for browser and Node.\+js.

The source code of this project can be cloned from the \href{https://github.com/kurento/kurento-jsonrpc-js}{\texttt{ Git\+Hub repository}}.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_2db70faf1bc6fff7a492b47c9ddb92b78_autotoc_md597}{}\doxysubsection{\texorpdfstring{Installation instructions}{Installation instructions}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_2db70faf1bc6fff7a492b47c9ddb92b78_autotoc_md597}
Be sure to have installed \mbox{[}Bower\mbox{]} in your system\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{sudo\ npm\ install\ -\/g\ bower}

\end{DoxyCode}


To install the library, it\textquotesingle{}s recomended to do that from the \mbox{[}Bower repository\mbox{]} \+:


\begin{DoxyCode}{0}
\DoxyCodeLine{bower\ install\ kurento-\/jsonrpc}

\end{DoxyCode}


Alternatively, you can download the code using git and install manually its dependencies\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{git\ clone\ https://github.com/Kurento/kurento-\/jsonrpc-\/js-\/bower.git}
\DoxyCodeLine{cd\ kurento-\/jsonrpc-\/js-\/bower}
\DoxyCodeLine{bower\ install}

\end{DoxyCode}

Kurento is an open source software project providing a platform suitable for creating modular applications with advanced real-\/time communication capabilities. For knowing more about Kurento, please visit the Kurento project website\+: \href{http://www.kurento.org}{\texttt{ http\+://www.\+kurento.\+org}}.

You can contribute to the Kurento community through bug-\/reports, bug-\/fixes, new code or new documentation. For contributing to the Kurento community, drop a post to the \href{https://groups.google.com/forum/\#!forum/kurento}{\texttt{ Kurento Public Mailing List}} providing full information about your contribution and its value. In your contributions, you must comply with the following guidelines


\begin{DoxyItemize}
\item You must specify the specific contents of your contribution either through a detailed bug description, through a pull-\/request or through a patch.
\item You must specify the licensing restrictions of the code you contribute.
\item For newly created code to be incorporated in the Kurento code-\/base, you must accept Kurento to own the code copyright, so that its open source nature is guaranteed.
\item You must justify appropriately the need and value of your contribution. The Kurento project has no obligations in relation to accepting contributions from third parties.
\item The Kurento project leaders have the right of asking for further explanations, tests or validations of any code contributed to the community before it being incorporated into the Kurento code-\/base. You must be ready to addressing all these kind of concerns before having your code approved.
\end{DoxyItemize}\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_2db70faf1bc6fff7a492b47c9ddb92b78_autotoc_md606}{}\doxysubsection{\texorpdfstring{Support}{Support}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_2db70faf1bc6fff7a492b47c9ddb92b78_autotoc_md606}

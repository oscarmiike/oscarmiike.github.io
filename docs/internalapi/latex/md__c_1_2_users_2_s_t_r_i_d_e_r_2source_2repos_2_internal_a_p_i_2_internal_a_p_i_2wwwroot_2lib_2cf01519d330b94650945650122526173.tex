 Follow the \href{http://contribute.jquery.org/style-guide/js/}{\texttt{ j\+Query style guide}}, with these exceptions\+:


\begin{DoxyItemize}
\item Use single quotes
\item Multiple var statements can be used to group variable declarations logically
\item Liberal spacing between parenthesis need not be used
\item Use \href{http://usejsdoc.org/\#JSDoc3_Tag_Dictionary}{\texttt{ jsdoc-\/style}} comments for functions
\end{DoxyItemize}

Add tests to test/bowercopy\+\_\+test.\+js. See the \href{https://github.com/caolan/nodeunit}{\texttt{ nodeunit README}} for documentation.

If you\textquotesingle{}re unfamiliar with grunt, \href{http://gruntjs.com/}{\texttt{ gruntjs.\+com}} is a good place to start. Follow instructions for installing grunt and see the Gruntfile for the tasks used.

Always run tests before submitting a pull request, unless it is just a documentation or copy change.

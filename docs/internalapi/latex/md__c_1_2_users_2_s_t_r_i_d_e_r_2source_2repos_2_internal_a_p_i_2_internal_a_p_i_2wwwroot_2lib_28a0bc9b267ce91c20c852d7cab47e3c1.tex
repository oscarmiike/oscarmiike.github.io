You can optionally specify a version (e.\+g. {\ttfamily 1.\+0.\+0}) or {\ttfamily latest}. ~\newline
 If you don\textquotesingle{}t specify a version you\textquotesingle{}ll get the \href{http://registry.npmjs.org/pageres/}{\texttt{ main entry}} containing all versions.

The version can also be in any format supported by the \href{https://www.npmjs.com/package/semver}{\texttt{ semver}} module. For example\+:


\begin{DoxyItemize}
\item {\ttfamily 1} -\/ get the latest {\ttfamily 1.\+x.\+x}
\item {\ttfamily 1.\+2} -\/ get the latest {\ttfamily 1.\+2.\+x}
\item {\ttfamily \texorpdfstring{$^\wedge$}{\string^}1.2.\+3} -\/ get the latest {\ttfamily 1.\+x.\+x} but at least {\ttfamily 1.\+2.\+3}
\item {\ttfamily \texorpdfstring{$\sim$}{\string~}1.2.\+3} -\/ get the latest {\ttfamily 1.\+2.\+x} but at least {\ttfamily 1.\+2.\+3}

Generates a pseudo-\/random \href{http://en.wikipedia.org/wiki/Globally_unique_identifier}{\texttt{ Globally Unique Identifier}} (v4).

Since the total number of GUIDs is 2\textsuperscript{122} the chance of generating the same value twice is negligible.

{\bfseries{Important\+:}} this method uses {\ttfamily Math.\+random} by default so the UUID isn\textquotesingle{}t {\itshape safe} (sequence of outputs can be predicted in some cases), check the \`{}random()\`{} documentation for more info on how to replace the default PRNG if you need extra safety or need {\itshape seeded} results.

{\bfseries{Important\+:}} No methods inside mout should call {\ttfamily Math.\+random()} directly, they all use {\ttfamily random/random} as a proxy, that way we can inject/replace the pseudo-\/random number generator if needed (ie. in case we need a seeded random or a better algorithm than the native one).\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_2fc1048cee0dd3b3519c52abeefef3efd_autotoc_md3703}{}\doxysubsection{\texorpdfstring{Replacing the PRNG}{Replacing the PRNG}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_internal_a_p_i_2_internal_a_p_i_2wwwroot_2lib_2fc1048cee0dd3b3519c52abeefef3efd_autotoc_md3703}
In some cases we might need better/different algorithms than the one provided by {\ttfamily Math.\+random} (ie. safer, seeded).

Because of licensing issues, file size limitations and different needs we decided to {\bfseries{not}} implement a custom PRNG and instead provide a easy way to override the default behavior. -\/ \href{https://github.com/millermedeiros/amd-utils/issues/99}{\texttt{ issue \#99}}

\doxysubsection{Detailed Description}
Retrieves log entries based on the provided Log\+Request parameters. Converts date ranges from UTC to local time. Filters logs by date range, log type, source, and search string. Handles pagination by checking if it is an initial request. Orders logs descending by created date. Limits results based on max\+Logs parameter.

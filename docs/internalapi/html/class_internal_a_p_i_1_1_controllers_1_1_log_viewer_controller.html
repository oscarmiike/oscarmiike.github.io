IActionResult InternalAPI.Controllers.LogViewerController.GetLogs ( [FromBody] LogRequest logRequest)

HTTP POST controller actions to retrieve log entries from various services. The actions accept a LogRequest model in the request body to specify the parameters for retrieving the logs. They call out to injected service dependencies to retrieve the actual log data. The log entries are returned in the response by rendering a partial view for display. Common validations and processing is done on the log request and response data.

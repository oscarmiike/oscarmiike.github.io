‍Returns true if the path appears to be relative.

Install

Install with npm:

$ npm install --save is-relative

Usage

var isRelative = require('is-relative');
console.log(isRelative('README.md'));
//=> true
console.log(isRelative('/User/dev/foo/README.md'));
//=> false

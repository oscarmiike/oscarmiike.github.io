Created by \href{https://github.com/jonschlinkert}{\texttt{ jonschlinkert}} and \href{https://github.com/doowb}{\texttt{ doowb}}, Enquirer is fast, easy to use, and lightweight enough for small projects, while also being powerful and customizable enough for the most advanced use cases.


\begin{DoxyItemize}
\item {\bfseries{Fast}} -\/ Loads in \texorpdfstring{$\sim$}{\string~}4ms (that\textquotesingle{}s about {\itshape 3-\/4 times faster than a \href{http://www.endmemo.com/sconvert/framespersecondframespermillisecond.php}{\texttt{ single frame of a HD movie}} at 60fps})
\item {\bfseries{Lightweight}} -\/ Only one dependency, the excellent \href{https://github.com/doowb/ansi-colors}{\texttt{ ansi-\/colors}} by \href{https://github.com/doowb}{\texttt{ Brian Woodward}}.
\item {\bfseries{Easy to implement}} -\/ Uses promises and async/await and sensible defaults to make prompts easy to create and implement.
\item {\bfseries{Easy to use}} -\/ Thrill your users with a better experience! Navigating around input and choices is a breeze. You can even create \href{examples/fun/countdown.js}{\texttt{ quizzes}}, or \href{examples/fun/record.js}{\texttt{ record}} and \href{examples/fun/play.js}{\texttt{ playback}} key bindings to aid with tutorials and videos.
\item {\bfseries{Intuitive}} -\/ Keypress combos are available to simplify usage.
\item {\bfseries{Flexible}} -\/ All prompts can be used standalone or chained together.
\item {\bfseries{Stylish}} -\/ Easily override semantic styles and symbols for any part of the prompt.
\item {\bfseries{Extensible}} -\/ Easily create and use custom prompts by extending Enquirer\textquotesingle{}s built-\/in prompts.
\item {\bfseries{Pluggable}} -\/ Add advanced features to Enquirer using plugins.
\item {\bfseries{Validation}} -\/ Optionally validate user input with any prompt.
\item {\bfseries{Well tested}} -\/ All prompts are well-\/tested, and tests are easy to create without having to use brittle, hacky solutions to spy on prompts or "{}inject"{} values.
\item {\bfseries{Examples}} -\/ There are numerous \mbox{[}examples\mbox{]}(examples) available to help you get started.
\end{DoxyItemize}

If you like Enquirer, please consider starring or tweeting about this project to show your support. Thanks!

{\bfseries{Prompts control how values are rendered and returned}}

Each individual prompt is a class with special features and functionality for rendering the types of values you want to show users in the terminal, and subsequently returning the types of values you need to use in your application.

{\bfseries{How can I customize prompts?}}

Below in this guide you will find information about creating custom prompts. For now, we\textquotesingle{}ll focus on how to customize an existing prompt.

All of the individual prompt classes in this library are exposed as static properties on Enquirer. This allows them to be used directly without using {\ttfamily enquirer.\+prompt()}.

Use this approach if you need to modify a prompt instance, or listen for events on the prompt.

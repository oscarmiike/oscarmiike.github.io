Return an array of concatenated elements by calling {\ttfamily fn(x, i)} for each element {\ttfamily x} and each index {\ttfamily i} in the array {\ttfamily xs}.

When {\ttfamily fn(x, i)} returns an array, its result will be concatenated with the result array. If {\ttfamily fn(x, i)} returns anything else, that value will be pushed onto the end of the result array.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m8b6f9b137bdbff6d79b147f7fb1bdef4_autotoc_md5583}{}\doxysection{\texorpdfstring{install}{install}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m8b6f9b137bdbff6d79b147f7fb1bdef4_autotoc_md5583}

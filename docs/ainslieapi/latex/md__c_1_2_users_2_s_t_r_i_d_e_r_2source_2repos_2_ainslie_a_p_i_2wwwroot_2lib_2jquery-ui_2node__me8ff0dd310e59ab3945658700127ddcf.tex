 \begin{quote}
Strip comments from JSON. Lets you use comments in your JSON files! \end{quote}
This is now possible\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\{}
\DoxyCodeLine{\ \ \ \ //\ rainbows}
\DoxyCodeLine{\ \ \ \ "{}unicorn"{}:\ /*\ ❤\ */\ "{}cake"{}}
\DoxyCodeLine{\}}

\end{DoxyCode}


It will replace single-\/line comments {\ttfamily //} and multi-\/line comments {\ttfamily /\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}/} with whitespace. This allows JSON error positions to remain as close as possible to the original source.

Also available as a \href{https://github.com/sindresorhus/gulp-strip-json-comments}{\texttt{ gulp}}/\href{https://github.com/sindresorhus/grunt-strip-json-comments}{\texttt{ grunt}}/\href{https://github.com/sindresorhus/broccoli-strip-json-comments}{\texttt{ broccoli}} plugin.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__me8ff0dd310e59ab3945658700127ddcf_autotoc_md4795}{}\doxysection{\texorpdfstring{Install}{Install}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__me8ff0dd310e59ab3945658700127ddcf_autotoc_md4795}

A better \href{https://github.com/pwnall/node-open}{\texttt{ node-\/open}}. Opens stuff like websites, files, executables. Cross-\/platform. \end{quote}

\item Actively maintained
\item Supports app arguments
\item Safer as it uses {\ttfamily spawn} instead of {\ttfamily exec}
\item Fixes most of the open {\ttfamily node-\/open} issues

Uses the command {\ttfamily open} on OS X, {\ttfamily start} on Windows and {\ttfamily xdg-\/open} on other platforms.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__mf0a6516ddee8d9870afc6fc5ec072c56_autotoc_md3858}{}\doxysubsection{\texorpdfstring{opn(target, \mbox{[}options\mbox{]})}{opn(target, \mbox{[}options\mbox{]})}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__mf0a6516ddee8d9870afc6fc5ec072c56_autotoc_md3858}
Returns a promise for the \href{https://nodejs.org/api/child_process.html\#child_process_class_childprocess}{\texttt{ spawned child process}}. You\textquotesingle{}d normally not need to use this for anything, but it can be useful if you\textquotesingle{}d like to attach custom event listeners or perform other operations directly on the spawned process.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__mf0a6516ddee8d9870afc6fc5ec072c56_autotoc_md3859}{}\doxysubsubsection{\texorpdfstring{target}{target}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__mf0a6516ddee8d9870afc6fc5ec072c56_autotoc_md3859}
{\itshape Required} ~\newline
 Type\+: {\ttfamily string}

The thing you want to open. Can be a URL, file, or executable.

Opens in the default app for the file type. Eg. URLs opens in your default browser.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__mf0a6516ddee8d9870afc6fc5ec072c56_autotoc_md3860}{}\doxysubsubsection{\texorpdfstring{options}{options}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__mf0a6516ddee8d9870afc6fc5ec072c56_autotoc_md3860}
Type\+: {\ttfamily object}\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__mf0a6516ddee8d9870afc6fc5ec072c56_autotoc_md3861}{}\doxyparagraph{\texorpdfstring{wait}{wait}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__mf0a6516ddee8d9870afc6fc5ec072c56_autotoc_md3861}
Type\+: {\ttfamily boolean} ~\newline
 Default\+: {\ttfamily true}

Wait for the opened app to exit before calling the {\ttfamily callback}. If {\ttfamily false} it\textquotesingle{}s called immediately when opening the app.

On Windows you have to explicitly specify an app for it to be able to wait.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__mf0a6516ddee8d9870afc6fc5ec072c56_autotoc_md3862}{}\doxyparagraph{\texorpdfstring{app}{app}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__mf0a6516ddee8d9870afc6fc5ec072c56_autotoc_md3862}
Type\+: {\ttfamily string}, {\ttfamily array}

Specify the app to open the {\ttfamily target} with, or an array with the app and app arguments.

The app name is platform dependent. Don\textquotesingle{}t hard code it in reusable modules. Eg. Chrome is {\ttfamily google chrome} on OS X, {\ttfamily google-\/chrome} on Linux and {\ttfamily chrome} on Windows.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__mf0a6516ddee8d9870afc6fc5ec072c56_autotoc_md3863}{}\doxysection{\texorpdfstring{Related}{Related}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__mf0a6516ddee8d9870afc6fc5ec072c56_autotoc_md3863}

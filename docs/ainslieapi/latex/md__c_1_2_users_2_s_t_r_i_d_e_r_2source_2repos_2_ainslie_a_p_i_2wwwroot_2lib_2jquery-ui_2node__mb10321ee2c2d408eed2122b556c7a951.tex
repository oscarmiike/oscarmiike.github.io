Sem\+Ver 5.\+3.\+0

A Java\+Script implementation of the \href{http://semver.org/}{\texttt{ http\+://semver.\+org/}} specification Copyright Isaac Z. Schlueter

Usage\+: semver \mbox{[}options\mbox{]} $<$version$>$ \mbox{[}$<$version$>$ \mbox{[}...\mbox{]}\mbox{]} Prints valid versions sorted by Sem\+Ver precedence

Options\+: -\/r --range $<$range$>$ Print versions that match the specified range.

-\/i --increment \mbox{[}$<$level$>$\mbox{]} Increment a version by the specified level. Level can be one of\+: major, minor, patch, premajor, preminor, prepatch, or prerelease. Default level is \textquotesingle{}patch\textquotesingle{}. Only one version may be specified.

--preid $<$identifier$>$ Identifier to be used to prefix premajor, preminor, prepatch or prerelease version increments.

-\/l --loose Interpret versions and ranges loosely

Program exits successfully if any valid version satisfies all supplied ranges, and prints all satisfying versions.

If no satisfying versions are found, then exits failure.

Versions are printed in ascending order, so supplying multiple versions to the utility will just sort them.

\DoxyCodeLine{\#\#\ Ranges}
\DoxyCodeLine{}
\DoxyCodeLine{A\ \`{}version\ range`\ is\ a\ set\ of\ \`{}comparators`\ which\ specify\ versions}
\DoxyCodeLine{that\ satisfy\ the\ range.}
\DoxyCodeLine{}
\DoxyCodeLine{A\ \`{}comparator`\ is\ composed\ of\ an\ \`{}operator`\ and\ a\ \`{}version`.\ \ The\ set}
\DoxyCodeLine{of\ primitive\ \`{}operators`\ is:}
\DoxyCodeLine{}
\DoxyCodeLine{*\ \`{}<`\ Less\ than}
\DoxyCodeLine{*\ \`{}<=`\ Less\ than\ or\ equal\ to}
\DoxyCodeLine{*\ \`{}>`\ Greater\ than}
\DoxyCodeLine{*\ \`{}>=`\ Greater\ than\ or\ equal\ to}
\DoxyCodeLine{*\ \`{}=`\ Equal.\ \ If\ no\ operator\ is\ specified,\ then\ equality\ is\ assumed,}
\DoxyCodeLine{\ \ so\ this\ operator\ is\ optional,\ but\ MAY\ be\ included.}
\DoxyCodeLine{}
\DoxyCodeLine{For\ example,\ the\ comparator\ \`{}>=1.2.7`\ would\ match\ the\ versions}
\DoxyCodeLine{\`{}1.2.7`,\ \`{}1.2.8`,\ \`{}2.5.3`,\ and\ \`{}1.3.9`,\ but\ not\ the\ versions\ \`{}1.2.6`}
\DoxyCodeLine{or\ \`{}1.1.0`.}
\DoxyCodeLine{}
\DoxyCodeLine{Comparators\ can\ be\ joined\ by\ whitespace\ to\ form\ a\ \`{}comparator\ set`,}
\DoxyCodeLine{which\ is\ satisfied\ by\ the\ **intersection**\ of\ all\ of\ the\ comparators}
\DoxyCodeLine{it\ includes.}
\DoxyCodeLine{}
\DoxyCodeLine{A\ range\ is\ composed\ of\ one\ or\ more\ comparator\ sets,\ joined\ by\ \`{}||`.\ \ A}
\DoxyCodeLine{version\ matches\ a\ range\ if\ and\ only\ if\ every\ comparator\ in\ at\ least}
\DoxyCodeLine{one\ of\ the\ \`{}||`-\/separated\ comparator\ sets\ is\ satisfied\ by\ the\ version.}
\DoxyCodeLine{}
\DoxyCodeLine{For\ example,\ the\ range\ \`{}>=1.2.7\ <1.3.0`\ would\ match\ the\ versions}
\DoxyCodeLine{\`{}1.2.7`,\ \`{}1.2.8`,\ and\ \`{}1.2.99`,\ but\ not\ the\ versions\ \`{}1.2.6`,\ \`{}1.3.0`,}
\DoxyCodeLine{or\ \`{}1.1.0`.}
\DoxyCodeLine{}
\DoxyCodeLine{The\ range\ \`{}1.2.7\ ||\ >=1.2.9\ <2.0.0`\ would\ match\ the\ versions\ \`{}1.2.7`,}
\DoxyCodeLine{\`{}1.2.9`,\ and\ \`{}1.4.6`,\ but\ not\ the\ versions\ \`{}1.2.8`\ or\ \`{}2.0.0`.}
\DoxyCodeLine{}
\DoxyCodeLine{\#\#\#\ Prerelease\ Tags}
\DoxyCodeLine{}
\DoxyCodeLine{If\ a\ version\ has\ a\ prerelease\ tag\ (for\ example,\ \`{}1.2.3-\/alpha.3`)\ then}
\DoxyCodeLine{it\ will\ only\ be\ allowed\ to\ satisfy\ comparator\ sets\ if\ at\ least\ one}
\DoxyCodeLine{comparator\ with\ the\ same\ \`{}[major,\ minor,\ patch]`\ tuple\ also\ has\ a}
\DoxyCodeLine{prerelease\ tag.}
\DoxyCodeLine{}
\DoxyCodeLine{For\ example,\ the\ range\ \`{}>1.2.3-\/alpha.3`\ would\ be\ allowed\ to\ match\ the}
\DoxyCodeLine{version\ \`{}1.2.3-\/alpha.7`,\ but\ it\ would\ *not*\ be\ satisfied\ by}
\DoxyCodeLine{\`{}3.4.5-\/alpha.9`,\ even\ though\ \`{}3.4.5-\/alpha.9`\ is\ technically\ "{}greater}
\DoxyCodeLine{than"{}\ \`{}1.2.3-\/alpha.3`\ according\ to\ the\ SemVer\ sort\ rules.\ \ The\ version}
\DoxyCodeLine{range\ only\ accepts\ prerelease\ tags\ on\ the\ \`{}1.2.3`\ version.\ \ The}
\DoxyCodeLine{version\ \`{}3.4.5`\ *would*\ satisfy\ the\ range,\ because\ it\ does\ not\ have\ a}
\DoxyCodeLine{prerelease\ flag,\ and\ \`{}3.4.5`\ is\ greater\ than\ \`{}1.2.3-\/alpha.7`.}
\DoxyCodeLine{}
\DoxyCodeLine{The\ purpose\ for\ this\ behavior\ is\ twofold.\ \ First,\ prerelease\ versions}
\DoxyCodeLine{frequently\ are\ updated\ very\ quickly,\ and\ contain\ many\ breaking\ changes}
\DoxyCodeLine{that\ are\ (by\ the\ author's\ design)\ not\ yet\ fit\ for\ public\ consumption.}
\DoxyCodeLine{Therefore,\ by\ default,\ they\ are\ excluded\ from\ range\ matching}
\DoxyCodeLine{semantics.}
\DoxyCodeLine{}
\DoxyCodeLine{Second,\ a\ user\ who\ has\ opted\ into\ using\ a\ prerelease\ version\ has}
\DoxyCodeLine{clearly\ indicated\ the\ intent\ to\ use\ *that\ specific*\ set\ of}
\DoxyCodeLine{alpha/beta/rc\ versions.\ \ By\ including\ a\ prerelease\ tag\ in\ the\ range,}
\DoxyCodeLine{the\ user\ is\ indicating\ that\ they\ are\ aware\ of\ the\ risk.\ \ However,\ it}
\DoxyCodeLine{is\ still\ not\ appropriate\ to\ assume\ that\ they\ have\ opted\ into\ taking\ a}
\DoxyCodeLine{similar\ risk\ on\ the\ *next*\ set\ of\ prerelease\ versions.}
\DoxyCodeLine{}

All methods and classes take a final {\ttfamily loose} boolean argument that, if true, will be more forgiving about not-\/quite-\/valid semver strings. The resulting output will always be 100\% strict, of course.

Strict-\/mode Comparators and Ranges will be strict about the Sem\+Ver strings that they parse.


\begin{DoxyItemize}
\item {\ttfamily valid(v)}\+: Return the parsed version, or null if it\textquotesingle{}s not valid.
\item {\ttfamily inc(v, release)}\+: Return the version incremented by the release type ({\ttfamily major}, {\ttfamily premajor}, {\ttfamily minor}, {\ttfamily preminor}, {\ttfamily patch}, {\ttfamily prepatch}, or {\ttfamily prerelease}), or null if it\textquotesingle{}s not valid
\begin{DoxyItemize}
\item {\ttfamily premajor} in one call will bump the version up to the next major version and down to a prerelease of that major version. {\ttfamily preminor}, and {\ttfamily prepatch} work the same way.
\item If called from a non-\/prerelease version, the {\ttfamily prerelease} will work the same as {\ttfamily prepatch}. It increments the patch version, then makes a prerelease. If the input version is already a prerelease it simply increments it.

Note that, since ranges may be non-\/contiguous, a version might not be greater than a range, less than a range, {\itshape or} satisfy a range! For example, the range {\ttfamily 1.\+2 \texorpdfstring{$<$}{<}1.\+2.\+9 \texorpdfstring{$\vert$}{|}\texorpdfstring{$\vert$}{|} \texorpdfstring{$>$}{>}2.\+0.\+0} would have a hole from {\ttfamily 1.\+2.\+9} until {\ttfamily 2.\+0.\+0}, so the version {\ttfamily 1.\+2.\+10} would not be greater than the range (because {\ttfamily 2.\+0.\+1} satisfies, which is higher), nor less than the range (since {\ttfamily 1.\+2.\+8} satisfies, which is lower), and it also does not satisfy the range.

If you want to know if a version satisfies or does not satisfy a range, use the {\ttfamily satisfies(version, range)} function.

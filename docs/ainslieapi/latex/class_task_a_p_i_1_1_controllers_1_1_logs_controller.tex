Checks for alerts across all configured API sources as defined in Log\+Notifications\+Settings. 

\begin{DoxyReturn}{Returns}
Returns a status code 200 with a message and list of processed sources if all API sources are processed successfully. Returns a status code 500 along with a message if Log\+Notifications\+Settings are not found or any API sources fail to process. 
\end{DoxyReturn}


Iterates through all API sources specified in the Log\+Notifications\+Settings configuration. Each source is processed to check for alerts. Logs the exception and adds the source to a list of failed processes if any source fails to process. 

$<$response code="{}200"{}$>$All API sources processed for alerts. Includes environment name and lists processed sources.$<$/response$>$ $<$response code="{}500"{}$>$Indicates either Log\+Notifications\+Settings not found or some API sources failed to process.

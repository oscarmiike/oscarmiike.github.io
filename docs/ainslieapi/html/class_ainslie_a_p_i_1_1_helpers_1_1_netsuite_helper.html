GreenIdCustomer greenIDCustomer )

Creates a request object for creating or updating a customer in Netsuite based on Green ID verification data.

Parameters
branchThe branch identifier, used for setting the subsidiary in Netsuite.
greenIDCustomerThe customer's Green ID verification details.
Returns
A populated NetsuiteCustomerRequest object.

Maps Green ID customer data to the Netsuite customer request format, preparing data for integration with Netsuite CRM. Includes handling for personal details, contact information, and address, with specific attention to Netsuite's data structure.

IActionResult AinslieAPI.Controllers.ProductsController.PriceList ( )

Retrieves a list of current product prices.

Returns
An IActionResult containing the product pricing information or an error message.

This endpoint requires authorisation and is accessible only to users with specific roles. It fetches product pricing data using the InternalApiService and returns this data within an HTTP response. Logging is performed to capture the request context and authenticated user's details.

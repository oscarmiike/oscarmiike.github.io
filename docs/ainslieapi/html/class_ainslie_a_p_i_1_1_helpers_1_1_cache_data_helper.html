List< AssetPrice > AinslieAPI.Helpers.CacheDataHelper.RefreshPriceappAssets ( )
private

Refreshes the cache of asset prices by fetching new data if the current cache is expired or not present.

Returns
A list of AssetPrice, either newly fetched or existing, depending on cache validity.

Performs a thread-safe update to the asset prices cache using a padlock mechanism to prevent concurrent refreshes. This method:

  • Checks if an asset list cache exists and is current. If not, initiates a refresh.
  • If cache is absent or expired, creates a new cache item or updates the existing one with fresh data from the internal API service.
  • Applies a lock to ensure that refresh operations do not overlap, maintaining data integrity.
  • Updates the cache expiry based on application settings to determine the next refresh timing.

Catches and handles exceptions, ensuring the method returns a valid list of asset prices under all circumstances.

‍Returns true if the given string or array ends with suffix using strict equality for comparisons.

Install

Install with npm:

npm i ends-with --save-dev

Run benchmarks

This uses the fasted version of 9 different implementations that were benchmarked.

node benchmark

Run tests

npm test

Usage

var endsWith = require('ends-with');
endsWith('abc', 'c');
//=> true
endsWith(['a', 'b', 'c'], 'c');
//=> true
endsWith(['abc', 'abc', 'abc'], 'c');
//=> false
endsWith(['abc', 'abc', 'abc'], 'c');
//=> false
endsWith(['a', 'b', 42], 42)
//=> true
endsWith(['a', 'b', 42], '42')
//=> false

Initiates a new customer verification process, accommodating both GET and POST requests.

Parameters
branchIdentifies the branch for which verification is requested.
useridSpecifies the user's ID for verification, optional.
Returns
An IActionResult indicating the operation's outcome.

This process involves:

  • Logging the request details.
  • Retrieving and validating the customer's data from Green ID.
  • Updating or adding the customer's information to Netsuite through Internal API.

Returns OK if the process is successful, otherwise logs the error details.

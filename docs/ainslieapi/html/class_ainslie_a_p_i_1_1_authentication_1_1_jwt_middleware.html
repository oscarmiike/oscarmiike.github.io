Processes each HTTP request, attempting to validate JWT tokens and attach authenticated users to the context.

Parameters

jwtUtilsUtility class for JWT token validation.
Returns
A Task representing the asynchronous operation.

This middleware extracts the JWT token from the Authorisation header, validates it, and, if successful, attaches the decoded AuthenticatedUser object to the context for downstream use. Note: This does not integrate with .NET Core's built-in authentication system and considers no user as authenticated by default. Future improvements may include integrating JWT authentication middleware to work alongside existing authentication mechanisms.

static AuthenticatedUser AinslieAPI.Helpers.UserHelper.GetAuthenticatedUser ( ClaimsPrincipal user)
static

Extracts and constructs an AuthenticatedUser object from the security claims of the currently authenticated user.

Parameters
userThe ClaimsPrincipal representing the current user.
Returns
An AuthenticatedUser populated with details from the user's claims, or null if the user is not authenticated.

This method parses claims to retrieve essential user information such as name, email address, profile image, and roles. It ensures that only authenticated users' details are processed.

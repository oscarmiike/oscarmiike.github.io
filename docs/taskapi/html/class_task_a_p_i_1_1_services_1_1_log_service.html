List< LogEntry > TaskAPI.Services.LogService.FetchLogs ( LogRequest logRequest)

Fetches log entries based on specified criteria from the database.

Parameters
logRequestThe criteria for fetching logs, which includes the date range, log type, log source, and a search string for filtering logs.

This method applies the specified filter criteria to retrieve relevant log entries. It supports filtering by:

  • Date range: Logs within the specified start and end dates are fetched.
  • Log type: Only logs matching the specified log types (if any) are included.
  • Log source: Logs originating from specified sources are selected.
  • Search string: Logs containing the specified text in their data are fetched.

Additional logic is applied to adjust for time zones and to handle pagination and ordering, ensuring that the most relevant logs are retrieved efficiently.

Returns
A list of LogEntry objects that match the specified criteria. The logs are returned in descending order by creation date, ensuring that the most recent logs are returned first.

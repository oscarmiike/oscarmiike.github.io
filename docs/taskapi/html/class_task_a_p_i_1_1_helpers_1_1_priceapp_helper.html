bool TaskAPI.Helpers.PriceappHelper.Error

Creates a live pricesheet PDF based on the current time window.

Parameters
ignoreTimeA flag indicating whether to ignore time window restrictions for debugging purposes.
Returns
A tuple indicating the success status and result details of the operation.

This method generates a live pricesheet PDF based on the current time window.

It checks whether the current time falls within the daily or monthly time windows and proceeds accordingly.

If the ignoreTime parameter is set to true, the method ignores time window restrictions for debugging purposes.

\doxysubsection{Detailed Description}
Provides functionality to interact with the Alchemy API for blockchain data retrieval. 

This service is responsible for accessing blockchain data through the Alchemy API. It utilises the Nethereum.\+Web3 library to connect to the Ethereum blockchain and execute smart contract functions.

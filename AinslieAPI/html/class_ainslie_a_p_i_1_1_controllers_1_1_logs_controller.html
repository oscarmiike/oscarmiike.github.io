IActionResult AinslieAPI.Controllers.LogsController.GetLogs ( [FromBody] LogRequest logRequest)

Retrieves logs based on specified criteria, including time range and filters.

Parameters
logRequestDetails of the log request, such as time range, type, source, and search string.
Returns
An IActionResult containing logs matching the criteria.

This method enables both initial and follow-up log retrieval, featuring:

  • Time conversions from Unix/UTC to local, ensuring accurate time frame queries.
  • Filter application by log type, source, and search string for targeted searches.
  • Pagination via 'initial' flag and 'startHour', adjusting queries for specific log segments.
  • Descending order sorting by creation date for recent logs prioritisation.
  • Adaptation to request 'maxLogs' limit for controlled data retrieval.

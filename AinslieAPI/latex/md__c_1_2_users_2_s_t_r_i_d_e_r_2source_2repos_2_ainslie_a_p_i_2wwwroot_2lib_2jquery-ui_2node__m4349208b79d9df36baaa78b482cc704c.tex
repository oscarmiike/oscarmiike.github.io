Check if the path is executable. If no callback provided, and a global {\ttfamily Promise} object is available, then a Promise will be returned.

Will raise whatever errors may be raised by {\ttfamily fs.\+stat}, unless {\ttfamily options.\+ignore\+Errors} is set to true.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m4349208b79d9df36baaa78b482cc704c_autotoc_md3029}{}\doxysubsection{\texorpdfstring{{\ttfamily isexe.\+sync(path, \mbox{[}options\mbox{]})}}{{\ttfamily isexe.\+sync(path, \mbox{[}options\mbox{]})}}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m4349208b79d9df36baaa78b482cc704c_autotoc_md3029}

Add this to your project\textquotesingle{}s {\ttfamily Gruntfile.\+js} gruntfile\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{grunt.loadNpmTasks('grunt-\/compare-\/size');}

\end{DoxyCode}


Then add "{}grunt-\/compare-\/size"{} to your package.\+json dependencies.

Then install the plugin with\+: {\ttfamily npm install}

The name to use in your own task definitions is {\ttfamily compare\+\_\+size} (with an underscore).\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m42024a11125e499abefe2007620420f3_autotoc_md8734}{}\doxysection{\texorpdfstring{Documentation}{Documentation}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m42024a11125e499abefe2007620420f3_autotoc_md8734}

To run a size comparison\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{grunt\ compare\_size}

\end{DoxyCode}


To get compressed comparisons of a single file\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{grunt\ compare\_size::<file>}

\end{DoxyCode}


To add a labeled measurement to the saved list of measurements\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{grunt\ compare\_size:add:<label>:...}

\end{DoxyCode}


To remove a labeled measurement\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{grunt\ compare\_size:remove:<label>:...}

\end{DoxyCode}


To view all saved measurements\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{grunt\ compare\_size:list}

\end{DoxyCode}


To clear out all saved measurements\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{grunt\ compare\_size:empty}

\end{DoxyCode}

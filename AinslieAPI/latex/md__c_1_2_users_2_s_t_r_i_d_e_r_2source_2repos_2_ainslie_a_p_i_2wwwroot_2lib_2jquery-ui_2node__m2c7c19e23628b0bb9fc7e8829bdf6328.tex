This can be used as a polyfill for the native node.\+js {\ttfamily path.\+parse()} method, and it also adds a few properties\+:


\begin{DoxyItemize}
\item {\ttfamily path}\+: the original filepath
\item {\ttfamily is\+Absolute}\+: (getter) true if the given path is absolute
\item {\ttfamily absolute}\+: (getter) fully resolved, absolute filepath
\item {\ttfamily dirname}\+: alias for {\ttfamily dir}
\item {\ttfamily basename}\+: alias for {\ttfamily base}
\item {\ttfamily extname}\+: alias for {\ttfamily ext}
\item {\ttfamily stem}\+: alias for {\ttfamily name}
\end{DoxyItemize}

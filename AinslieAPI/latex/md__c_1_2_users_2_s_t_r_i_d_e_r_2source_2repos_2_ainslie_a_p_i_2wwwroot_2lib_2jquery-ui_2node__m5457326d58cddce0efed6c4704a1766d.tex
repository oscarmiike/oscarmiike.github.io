{\ttfamily requireg} tries to find modules in global locations which are not natively supported by the node.\+js \href{http://nodejs.org/docs/latest/api/modules.html\#modules_all_together}{\texttt{ module resolve algorithm}}.

Supported locations\+:


\begin{DoxyItemize}
\item \$\+HOME/node\+\_\+modules (instead of \$\+HOME/.node\+\_\+modules)
\item \$\+HOME/node\+\_\+libraries (instead of \$\+HOME/.node\+\_\+libraries)
\item \$\+HOME/node\+\_\+packages (specific of {\ttfamily requireg})
\item \$\+PREFIX/lib/node\+\_\+modules (instead of \$\+PREFIX/lib/node)
\item \$\+NODE\+\_\+\+MODULES (use the specific modules path environment variable)
\end{DoxyItemize}\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m5457326d58cddce0efed6c4704a1766d_autotoc_md4412}{}\doxysection{\texorpdfstring{Resolution priority}{Resolution priority}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m5457326d58cddce0efed6c4704a1766d_autotoc_md4412}

Loads the bower configuration from the configuration files.

Configuration is overwritten (after camelcase normalisation) with {\ttfamily overwrites} argument.

This method overwrites following environment variables\+:


\begin{DoxyItemize}
\item {\ttfamily HTTP\+\_\+\+PROXY} with {\ttfamily proxy} configuration variable
\item {\ttfamily HTTPS\+\_\+\+PROXY} with {\ttfamily https-\/proxy} configuration variable
\item {\ttfamily NO\+\_\+\+PROXY} with {\ttfamily no-\/proxy} configuration variable
\end{DoxyItemize}

It also clears {\ttfamily http\+\_\+proxy}, {\ttfamily https\+\_\+proxy}, and {\ttfamily no\+\_\+proxy} environment variables.

To restore those variables you can use {\ttfamily restore} method.\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m26835907d432df41063830a73129f65d_autotoc_md5301}{}\doxysubsubsection{\texorpdfstring{restore()}{restore()}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m26835907d432df41063830a73129f65d_autotoc_md5301}

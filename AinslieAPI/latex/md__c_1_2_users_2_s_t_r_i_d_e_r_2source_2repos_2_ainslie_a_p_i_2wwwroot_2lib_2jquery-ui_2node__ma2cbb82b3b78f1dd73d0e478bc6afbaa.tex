\begin{DoxyItemize}

\item {\ttfamily returns} \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\{String\}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+: Returns the detected filepath if it exists, otherwise returns {\ttfamily null}.

When using the {\ttfamily nocase} option, this library will attempt to detect the filepath with the following methods\+:


\begin{DoxyEnumerate}
\item Try to read all files in the {\ttfamily filepath} using {\ttfamily fs.\+readdir\+Sync}. If successful and {\ttfamily filepath} is a directory, return the {\ttfamily filepath}.
\item Try to read all files in the {\ttfamily filepath}\textquotesingle{}s directory using {\ttfamily fs.\+readdir\+Sync}. If successful, do case insensitive comparasions of the {\ttfamily filepath} to the files in {\ttfamily filepath}\textquotesingle{}s directory.

Javascript lacks complete string manipulation operations. This is an attempt to fill that gap. List of build-\/in methods can be found for example from \href{http://www.diveintojavascript.com/core-javascript-reference/the-string-object}{\texttt{ Dive Into Java\+Script}}. Originally started as an Underscore.\+js extension but is a full standalone library nowadays.

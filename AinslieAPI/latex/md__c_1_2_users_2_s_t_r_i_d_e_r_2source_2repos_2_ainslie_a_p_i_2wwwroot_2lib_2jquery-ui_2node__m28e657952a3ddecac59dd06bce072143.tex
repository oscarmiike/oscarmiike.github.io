Ajv generates code to turn JSON Schemas into super-\/fast validation functions that are efficient for v8 optimization.

Currently Ajv is the fastest and the most standard compliant validator according to these benchmarks\+:


\begin{DoxyItemize}
\item \href{https://github.com/ebdrup/json-schema-benchmark}{\texttt{ json-\/schema-\/benchmark}} -\/ 50\% faster than the second place
\item \href{https://github.com/pandastrike/jsck\#benchmarks}{\texttt{ jsck benchmark}} -\/ 20-\/190\% faster
\item \href{https://rawgit.com/zaggino/z-schema/master/benchmark/results.html}{\texttt{ z-\/schema benchmark}}
\item \href{https://cdn.rawgit.com/playlyfe/themis/master/benchmark/results.html}{\texttt{ themis benchmark}}
\end{DoxyItemize}

\begin{DoxyItemize}
\item Ajv implements JSON Schema \href{http://json-schema.org/}{\texttt{ draft-\/06/07/2019-\/09/2020-\/12}} standards (draft-\/04 is supported in v6)\+:
\begin{DoxyItemize}
\item all validation keywords (see \href{https://ajv.js.org/json-schema.html}{\texttt{ JSON Schema validation keywords}})
\item \href{https://github.com/OAI/OpenAPI-Specification/blob/master/versions/3.0.3.md}{\texttt{ Open\+API}} extensions\+:
\begin{DoxyItemize}
\item NEW\+: keyword \href{https://ajv.js.org/json-schema.html\#discriminator}{\texttt{ discriminator}}.
\item keyword \href{https://ajv.js.org/json-schema.html\#nullable}{\texttt{ nullable}}.
\end{DoxyItemize}
\item full support of remote references (remote schemas have to be added with {\ttfamily add\+Schema} or compiled to be available)
\item support of recursive references between schemas
\item correct string lengths for strings with unicode pairs
\item JSON Schema \href{https://ajv.js.org/guide/formats.html}{\texttt{ formats}} (with \href{https://github.com/ajv-validator/ajv-formats}{\texttt{ ajv-\/formats}} plugin).
\item \href{https://ajv.js.org/api.html\#api-validateschema}{\texttt{ validates schemas against meta-\/schema}}
\end{DoxyItemize}
\item NEW\+: supports \href{https://datatracker.ietf.org/doc/rfc8927/}{\texttt{ JSON Type Definition}}\+:
\begin{DoxyItemize}
\item all keywords (see \href{https://ajv.js.org/json-type-definition.html}{\texttt{ JSON Type Definition schema forms}})
\item meta-\/schema for JTD schemas
\item "{}union"{} keyword and user-\/defined keywords (can be used inside "{}metadata"{} member of the schema)
\end{DoxyItemize}
\item supports \href{https://ajv.js.org/guide/environments.html\#browsers}{\texttt{ browsers}} and Node.\+js 10.\+x -\/ current
\item \href{https://ajv.js.org/guide/managing-schemas.html\#asynchronous-schema-loading}{\texttt{ asynchronous loading}} of referenced schemas during compilation
\item "{}\+All errors"{} validation mode with \href{https://ajv.js.org/options.html\#allerrors}{\texttt{ option all\+Errors}}
\item \href{https://ajv.js.org/api.html\#validation-errors}{\texttt{ error messages with parameters}} describing error reasons to allow error message generation
\item i18n error messages support with \href{https://github.com/ajv-validator/ajv-i18n}{\texttt{ ajv-\/i18n}} package
\item \href{https://ajv.js.org/guide/modifying-data.html\#removing-additional-properties}{\texttt{ removing-\/additional-\/properties}}
\item \href{https://ajv.js.org/guide/modifying-data.html\#assigning-defaults}{\texttt{ assigning defaults}} to missing properties and items
\item \href{https://ajv.js.org/guide/modifying-data.html\#coercing-data-types}{\texttt{ coercing data}} to the types specified in {\ttfamily type} keywords
\item \href{https://ajv.js.org/guide/user-keywords.html}{\texttt{ user-\/defined keywords}}
\item additional extension keywords with \href{https://github.com/ajv-validator/ajv-keywords}{\texttt{ ajv-\/keywords}} package
\item \href{https://ajv.js.org/guide/combining-schemas.html\#data-reference}{\texttt{ \$data reference}} to use values from the validated data as values for the schema keywords
\item \href{https://ajv.js.org/guide/async-validation.html}{\texttt{ asynchronous validation}} of user-\/defined formats and keywords
\end{DoxyItemize}\hypertarget{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m28e657952a3ddecac59dd06bce072143_autotoc_md11590}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{md__c_1_2_users_2_s_t_r_i_d_e_r_2source_2repos_2_ainslie_a_p_i_2wwwroot_2lib_2jquery-ui_2node__m28e657952a3ddecac59dd06bce072143_autotoc_md11590}
